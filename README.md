# Stat Roller

This NPM package generates a list of stats for tabletop gaming characters for D&D. It uses the following two methods.

- Standard Array
- 4d6 and drop the lowest

## Dependencies

- NPM 6.14.11
- Node v14.16.0

## Setup

- Clone this repo
- `cd ./stat-roller`
- `npm i -g .`
- `stat-roller`

## Expected Output

```
Standard Array: [ 15, 14, 13, 12, 10, 8 ]
4d6: [ 13, 12, 13, 13, 15, 17 ]
```
